# Counter App

A simple and elegant counter application built with React.

## Features

- **Increment**: Increase the counter by 1
- **Decrement**: Decrease the counter by 1
- **Reset**: Reset the counter to 0
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradient background with smooth animations

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Navigate to the counter-app directory:
```bash
cd counter-app
```

2. Install dependencies:
```bash
npm install
```

### Running the Application

Start the development server:
```bash
npm start
```

The application will open in your browser at `http://localhost:3000`.

### Building for Production

Create an optimized production build:
```bash
npm run build
```

The build folder will contain the production-ready files.
