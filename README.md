## Projects

### 📝 Notes App (`/notes-app`)

A clean, modern notes-taking app built with React.

**Features:**
- Create, edit, and delete notes
- Search notes by title or content
- Notes are persisted to `localStorage`
- Word & character count in the editor
- Fully responsive three-panel layout (Sidebar · Note List · Editor)

**Tech Stack:** React 18, UUID, plain CSS

**Getting started:**
```bash
cd notes-app
npm install
npm start
```
